# 신용카드 사용자 세그먼트 분류 프로젝트 (Credit Card User Segmentation)

이 프로젝트는 대용량 신용카드 사용 데이터를 분석하고 전처리하여, 사용자의 특성에 따라 5개 세그먼트(A~E)로 분류하는 머신러닝 파이프라인을 구축하는 것을 목표로 합니다.

## 조원
* **이건욱** : 파일을 전처리한 뒤 불러오는 함수 만들어주기, 전처리 및 피처 엔지니어링 하기, 조원들이 각자 작업한 파일을 통합하기
* **양동녕** : 기타 전처리 및 모델링
* **안도겸** : 
* **유창엽** : 시상자 코드 리뷰 및 피쳐 엔지니어링

## 📋 프로젝트 개요
* **데이터셋**: 회원정보, 신용정보, 승인매출정보 등 다양한 범주의 대규모 데이터 (Parquet 형식)
* **목적**: 사용자 특성 분석을 통한 5개 그룹(Segment A~E) 예측
* **주요 특징**: 대용량 데이터 처리 최적화 및 앙상블 모델링

## 🛠 기술 스택
* **언어**: Python
* **데이터 분석**: Pandas, NumPy, PyArrow
* **시각화**: Matplotlib, Seaborn
* **머신러닝**: XGBoost, LightGBM, CatBoost
* **하이퍼파라미터 최적화**: Optuna

## 🚀 프로젝트 주요 단계



### 1. 데이터 로드 및 병합
* 여러 폴더에 분산된 Parquet 파일을 효율적으로 로드하고 통합하는 `make_df` 함수를 구현하였습니다.

### 2. 데이터 최적화 (Memory Optimization)
* 약 240만 건의 대규모 데이터 처리를 위해 수치형 데이터를 적절한 타입으로 다운캐스팅하여 메모리 사용량을 절감하였습니다.

### 3. 피처 엔지니어링 (Feature Engineering)
* 도메인 지식을 바탕으로 '카드 즉시 가용 비율', '회원 이용가능 점수' 등 예측 성능을 높일 수 있는 파생 변수를 생성하였습니다.
* 시각화 도구를 통해 세그먼트별 주요 피처의 분포와 상관관계를 분석하였습니다.

### 4. 모델 구축 및 앙상블
* XGBoost, LightGBM, CatBoost 세 가지 부스팅 모델을 구축하였습니다.
* Optuna를 통해 각 모델의 최적 하이퍼파라미터를 탐색하였습니다.
* **Soft Voting 앙상블**: 각 모델의 예측 확률에 가중치(XGB 0.4, LGB 0.3, CAT 0.3)를 부여하여 최종 세그먼트를 결정하였습니다.
